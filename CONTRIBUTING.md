## Development Setup

1. Clone the repository:
git clone https://github.com/varshavskiyTymur/sb-content-creator.git
cd sb-content-creator2. Install dependencies:
npm install3. Build the project:
npm run build## Making Changes

1. Create a new branch:
git checkout -b feature/your-feature-name2. Make your changes
3. Build and test locally
4. Commit your changes:
git commit -m "feat: add new feature"5. Push and create a Pull Request

## Commit Convention

We use [Conventional Commits](https://www.conventionalcommits.org/):

- `feat:` - New feature
- `fix:` - Bug fix
- `docs:` - Documentation changes
- `refactor:` - Code refactoring
- `test:` - Adding tests
- `chore:` - Maintenance tasks
